# Moe's PC Parts

This is a COMP 3340 PHP and MySQL website for selling computer parts.

## Installation Documentation

- Install XAMPP.
- Start Apache.
- Start MySQL.
- Copy the project into htdocs.
- Open phpMyAdmin.
- Create a database named moes_pc_parts.
- Import the SQL database file.
- Check includes/database.php.
- Open localhost in a web browser.
