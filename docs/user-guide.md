# User Guide

## Register

Use Register to make a new account.

## Login

Use Login to enter the website.

## Browse Products

Use Products to see computer parts.

## Shopping Cart

Use the cart to view added products.

## Checkout

Use Checkout to place an order.

## Order History

Use Order History to see past orders.

## Profile

Use Profile to see account details.

## Themes

Use theme links to change the website colours.

## Help Pages

Use Help to read simple instructions.

## Statistics

Use Statistics to see product counts by category.

## Location

Use Location to see the store address.
